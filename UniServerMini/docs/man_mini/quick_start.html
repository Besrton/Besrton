Quick Start Guide

The Uniform Server PHP MiniServer’s default configuration is set to allow only local access; users on your network or the Internet cannot access your server. This allows you to develop and view your website locally in a secure environment.

This guide shows you how to run the server as standard programs. When run as standard program, nothing is installed to the Windows registry. The server is therefore fully portable.

Note: Interchange UniMiniServer and UniMicroServer as appropriate for your server variant.

Starting the Server

The PHP MiniServer's folder structure is shown on right.
  • A) Navigate to the UniServerMini folder.
    Locate the file phpMiniServer1.exe
    Note: File is common to both server variants.
  • B) Double click on file phpMiniServer1.exe
  • C) This starts The PHP MiniServer menu see image on right.
  • D) Click Start Server button. Starts server
  • If challenged by your firewall, allow access
  • Note Buttons/Indicator change state.
  • The Start menu button toggles displaying Stop Server.
    Status indicator changes to green indicating server running.
  • Buttons H) and I) become enabled:
    • H) Click button www displays index page contained in root folder www
    • I) Click button phpInfo displays php information page

Note: There are various start-up conditions. If you receive a message unable to connect and are provided an option from your browser to try again, wait until all disk activity completes and click the "try again" button. Alternatively, wait a reasonable time and enter the following into your browser: http://localhost/index.php or http://localhost/us_phpinfo.php

If you do not have the VC9 library, when you attempt to start the Server, you will get a runtime error popup. See VC9 Libraries to resolve this.

Root folder www - Test

The root folder www contains a very simple web-site consisting of a single page named index.php.

H) Clicking menu button www displays this page confirming that PHP Server is serving pages from this folder.

Add your Website or test pages

J) Copy your Website or test pages into the root folder www.

Before doing this with the exception of file favicon.ico you can delete the content of this root folder.

  • favicon.ico - This is a small image file displayed in a browser next to a Web page name.
    You can replace this with your own file. The name is important and should always be favicon.ico

View pages

The default server name is localhost. Use this to access pages as follows:

Index Pages

To access index pages (such as index.htm, index.html, index.php) type the following into your browser.

http://localhost/

Specific pages

Display a specific page by appending its name to the server name. For example, if your test page is named test.php, then type the following into your browser:

http://localhost/test.php

The PHP server automatically searches for index pages and displays the first one it finds. If it cannot find an index page, the following message is displayed: "The requested resource / was not found on this server."

Stopping the Server

Note: The server control button toggles between "Start Server" and "Stop Server".

Stop Server:
  • D) Click Stop Server button. Stops the server
  • The status indicator changes to red indicating server stopped.
Close Menu:
  • Click Window Close button (cross top right). Stops server menu utility.

Note: Closing the menu does not change servers’ state.
On restarting the menu, current server state is picked up and displayed.
