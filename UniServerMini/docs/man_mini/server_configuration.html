Server Configuration

The PHP MiniServer uses a configuration file to store settings. These are either manually entered by editing the file or automatically updated using the configuration menu.

This page covers the configuration menu. From this menu the server command window can be enabled allowing server activity to be observed or disabled (hidden) for silent operation. Server root folder and router file is selected from their corresponding popup menus allowing paths to be easily set.

Similarly a popup menu provides quick page assignment to the www button. Before committing to a new server-port it can be checked using a single button click. The default PHP configuration is selectable from one of three files, each file can be opened in notepad for editing.

Server configuration menu

The server configuration menu is shown on the right; at first glance looks a little bewildering however it is easy to use.

Tip: Restore default settings by clicking all default buttons, set and check port to use, click save configuration button.
For the default settings to become effective restart the server.

Change Server port:
  • A) Enter port to use.
  • B) Click "Check port" button. If in use set a new port.
Select PHP configuration file:
  • E) F) G) Click a radio button to select the PHP ini file to use.
  • I) This button restores default value and sets radio button accordingly.
  • Note: Use edits buttons to edit corresponding file in notepad.
Select root folder:
  • J) Click "Select" button. From the popup select folder to use.
    • Or click "Default" button to restore defaults.
Select router file:
  • K) Click "Select" button. From the popup select file to use.
    • Or click "Default" button to restore defaults.
Select www page to view in default browser:
  • L) Click "Select" button. From the popup select file to use.
    • Or click "Default" button to restore defaults.
    • Note: File selected is converted to a relative URL.
Hide server command window:
  • M) Checked hide window. Unchecked display window.
Save configuration:
  • C) Click "Save configuration" button. For the new settings to become effective restart the server.

Note: Server default access is restricted to the local machine this is changeable in configuration file
